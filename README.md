To run this app:

1. `mkdir music-list`
1. `cd music-list`
1. Make a virtual env ( called mine 'music-env' )
1. activate your virtual env ( `source music-env/bin/activate`)
1. clone this repo
1. cd into the repo
1. `pip install -r requirements.txt`
1. `python manage.py migrate`
1. `python manage.py runserver`
